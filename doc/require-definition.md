# 要求定義

- next と golang を使ったブログサイトを作る
- jamstack にする
- reveal.js を読み込めるようにする
- markdown を読み込めるようにする

## 要求定義詳細

- ブログを markdown 形式、reveal.js 形式で投稿できるようにする
- 下書き機能をつける
- プレビュー機能をつける
- リッチエディターにする
- マークダウンエディターがある
- サジェスト機能をつける
- 画像の投稿をできるようにする
- リンクに大してカード、タイトル、生リンクを選べるようにする
- シェアボタンをつける（line, twitter, facebook, はてぶ）
- 投稿に対してコメントできるようにする
- コメントに対してコメントできるようにする
- adsense をつける
